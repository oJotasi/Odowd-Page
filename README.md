# Odow-dPage

Aplicación web desarrollada con Django enfocada en la gestión y visualización de información mediante una interfaz dinámica e interactiva.

---

## Descripción

Odow-dPage es un proyecto desarrollado utilizando Django como framework principal para el backend. La aplicación integra herramientas de procesamiento y visualización de datos, formularios dinámicos y administración de contenido mediante una arquitectura modular basada en el patrón MVT de Django.

El proyecto fue desarrollado con el objetivo de fortalecer conocimientos en desarrollo web full stack, manejo de bases de datos, integración de librerías externas y organización de proyectos profesionales con Python.

---

## Tecnologías utilizadas

- Python
- Django
- HTML5
- CSS3
- Bootstrap
- JavaScript
- Pandas
- Matplotlib
- Django Crispy Forms
- SQLite
- Git & GitHub

---

## Funcionalidades principales

- Desarrollo backend utilizando Django.
- Formularios dinámicos con Django Crispy Forms.
- Diseño responsive utilizando Bootstrap.
- Procesamiento de datos con Pandas.
- Generación de gráficos y visualizaciones con Matplotlib.
- Panel de administración mediante Django Admin.
- Manejo de archivos estáticos y multimedia.
- Arquitectura modular mediante aplicaciones Django.
- Routing y renderizado dinámico de templates.
